Linuxmuster-webui7’s developer documentation¶

About¶

Welcome in this API reference documentation !

Linuxmuster-webui7 is a part of the linuxmuster.net’s project which provides a complete environment for school network management.

You can find the whole description of the install process on the official documentation page of the project.

In particularly, some appliances (Proxmox, XCP-ng and KVM) are prepared in order to permit a fast deployment for test use.

Ajenti¶

Linuxmuster-webui7 is based on the server admin panel Ajenti and only adds some plugins for a complete integration into a linuxmuster.net environment. The plugins are written in Python and AngularJS.

If you want to learn or contribute to this project, you need first to be ease with Ajenti’s install and dev documentation.

This documentation only lists an API reference for the specific plugins in linuxmuster.net.
